# MindCare-AI
Mental Health Awareness Web App using Python

## Project Overview
MindCare AI is a Python-based mental health awareness web application
designed for students. It uses indirect questions to help users reflect
on their emotional well-being and provides helpful resources and mentor
guidance.

## Features
- Emotional check-in using indirect questions
- Student well-being dashboard
- Mental health resources
- Mentor support section
- Simple and user-friendly interface

## Technology Used
- Python
- Streamlit

## How to Run the Project
1. Install Streamlit:
   pip install streamlit
2. Run the app:
   streamlit run app.py

## Purpose
This project is created for Class 12 Artificial Intelligence (Code 843)
practical examination.
